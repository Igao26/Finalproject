O que é o Programa Nagios

Nagios é uma popular aplicação de monitoramento de rede de código aberto distribuída sob a licença GPL. Ele pode monitorar tanto hosts quanto serviços, alertando quando ocorrerem problemas e também quando os problemas são resolvidos. O Nagios foi originalmente criado sob o nome de Netsaint, foi escrito e é atualmente mantido por Ethan Galstad, junto com uma equipe de desenvolvedores que ativamente mantém plugins oficiais e não-oficiais. Nagios primeiramente foi escrito para o sistema operacional Linux, mas pode rodar em outros Unix-like também.

Pra que serve o Nagios?

O Nagios serve para monitorar a rede e verificar constantemente a disponibilidade do serviço,sendo local ou remoto e avisa por meio de e-mail ou celular sobre o problema ocorrido,sendo possível obter relatórios de disponibilidade e configurar ações corretivas para os problemas ocorridos na rede, já que possui uma interface na qual permite o monitoramento via WEB e Browser, possuindo configurações de alertas e permitindo que tome ações corretivas para um erro eminente.

Exemplificando o Nagios

O Nagios pode ser controlado a partir de vários comandos que ele possui, dentre eles tem o comando check_ping, sendo um plug-in usado para verificar a saída do ping de um servidor remoto.É possível utilizar outros códigos junto do check_ping para especificar a ordem desejada.Por exemplo a opção -H que especifica o nome do Host ou o endereço IP do servidor para o qual gostaria de verificar a saída de comando ping.
